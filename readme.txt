== Description ==

With this plugin you add a click-to-call button to your responsive website in no-time. 

= Features =
* Simple installation
* Setup can be found under: settings > click-to-call menu

= Requirements =
This plugin requires PHP 5 or PHP 7 and WordPress 3.0 or greater. It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

= Contact =
* If you have any suggestion, feel free to email me at info@converzo.nl.
* See also website: http://converzo.nl

== Installation ==

1. Go to Plugins > Add New. 
2. Search "Click-to-Call Button by Converzo.nl" or upload file 
3. install and activate the WordPress Plugin.


== How to Use ==
Check settings under settings > click-to-call menu
